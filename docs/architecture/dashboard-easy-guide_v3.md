# 대시보드 보는 법 — 초등학생 가이드

> 이 문서는 http://localhost:8889 에서 보여주는 화면을 **초등학생도 이해할 수 있게** 설명해요. 기상청 날씨 예보랑 비슷하다고 생각하면 돼요!

---

## 대시보드가 뭐야?

**자동차 계기판**을 본 적 있나요?
운전할 때 속도계, 연료 게이지, 엔진 상태 이런 거 한눈에 보이잖아요.

이 대시보드도 똑같아요. **미국 주식 시장의 여러 정보**를 한 화면에 모아 보여주는 프로그램이에요.

왼쪽 메뉴에 **6개 페이지**가 있어요. 오늘은 그중에서 **"Index Forecast" (지수 예보)** 페이지를 자세히 배워볼 거예요.

## ★ Index Forecast — 주식 시장 날씨 예보 (메인!)

**기상청**이 "내일 비 올 확률 70%" 라고 알려주듯,
이 페이지는 **"5일 뒤에 미국 주식이 오를까? 내릴까?"** 를 AI가 예측해줘요.

### 1) 제목과 날짜

```

다음 5 거래일 방향 예측 · GRADIENTBOOSTING + TIMESERIESSPLIT CV
                                                    AS OF

```

- **Index Forecast** = "지수 예보" (주가 지수의 날씨 예보)
- **5 거래일 방향 예측** = 앞으로 5일 동안 오를지 내릴지 예측
- **AS OF 2026-04-05** = "2026년 4월 5일 기준 예보예요"
- **GRADIENTBOOSTING** = 예측에 쓴 AI 로봇 이름 (나무 여러 개를 모아 투표하는 로봇)

### 2) SPY 카드 — "미국 대표 주가 예보"

```
┌─ SPY ────────────────────────────────┐
│ SPY   ↓ BEARISH         HIGH 91.5%  │
│                                      │
│ PROBABILITY (UP)                8.5% │
│ [▓░░░░░░░░░░]                        │
│                                      │
│ PREDICTED RETURN   HORIZON           │
│ -3.135%            5D                │
└──────────────────────────────────────┘
```

**SPY**가 뭐예요?
- 미국에서 제일 큰 **회사 500개** 평균 주가예요.
- 한국으로 치면 **"코스피 200"** 같은 거예요.
- 애플, 마이크로소프트, 엔비디아, 구글 이런 대기업이 다 포함돼요.
- "미국 전체 주식 평균"을 대표해요.

---

#### BEARISH ↓ / BULLISH ↑ — 동물 이름!

주식 세계에서는 **동물 이름**으로 방향을 말해요.

| 단어 | 동물 | 뜻 | 왜? |
|------|-----|---|---|
| **BEARISH** | 곰 | 주식이 **떨어진다** | 곰이 발톱으로 아래로 할퀴니까 ↓ |
| **BULLISH** | 황소 | 주식이 **오른다** | 황소가 뿔로 위로 받아올리니까 ↑ |

화면에는 **"↓ BEARISH"** 라고 나와요.
→ "5일 뒤에 SPY가 **떨어질 것 같아요**" 라는 뜻이에요.

---

#### PROBABILITY (UP) 8.5% — "오를 확률"

막대 그래프 [▓░░░░░░░░░] 가 확률이에요.

- **8.5%** = 100번 중 **8번만 오른다**는 예측.
- 반대로 말하면 **92번은 떨어진다**!
- 막대가 짧으면 "오를 확률 낮음"
- 막대가 길면 "오를 확률 높음"

**퀴즈**: 오를 확률이 8.5%면, 떨어질 확률은 몇 %일까요?
정답: **91.5%** (100% - 8.5%)

---

#### PREDICTED RETURN -3.135% — "얼마나?"

"5일 뒤에 약 **3.1% 떨어질 거예요**" 라는 숫자 예측이에요.

예시로 계산해봐요:
- 오늘 SPY가 **100원**이면
- 5일 뒤 약 **96.9원**
- (100원 - 3.1원 = 96.9원)

**HORIZON 5D** = "5일(Day) 뒤까지 내다봐요"

---

#### HIGH 91.5% — "얼마나 확신해?"

이건 **예측 자신감 점수**예요.

| 점수 | 등급 | 뜻 | 우산 비유 |
|------|-----|---|---------|
| **70% 이상** | HIGH (높음) | "확신해!" | "우산 꼭 챙기세요!" |
| **60~70%** | MODERATE (중간) | "그럴 것 같아" | "혹시 모르니 챙겨요" |
| **60% 미만** | LOW (낮음) | "잘 모르겠어" | "챙길지 말지 잘 모름" |

91.5% HIGH이면 **아주 확신한다**는 뜻이에요.

### 3) QQQ 카드 — "미국 IT 주가 예보"

**QQQ**는 SPY 형제예요. 하지만 다른 점:
- QQQ는 **IT 회사 100개** 평균 주가예요.
- 애플, 구글, 아마존, 테슬라 같은 **기술 회사만 모음**.
- 한국으로 치면 **"코스닥"** 같은 거예요.

QQQ 카드도 SPY와 똑같은 형식이에요.
- 화면에는 **↓ BEARISH**, **HIGH 86.2%**, **-3.59%** 라고 나와요.
- → "5일 뒤 QQQ도 약 3.6% 떨어질 거 같아요 (86% 확신)"

### 4) KEY DRIVERS — "왜 그렇게 생각했어?" 5가지 이유

```
KEY DRIVERS (SPY)
yield_spread_proxy        BULLISH   0.365
████████████              
spy_price_vs_50ma_pct     BEARISH  -3.049
██████████████
spy_vol_trend_5d          BULLISH   0.03
█████████████
spy_return_1m             BEARISH  -4.277
███████████
xlu_relative_1m           BEARISH   2.309
█████████
```

**이게 뭐예요?**
- AI가 예측할 때 본 **가장 중요한 이유 5가지**예요.
- 마치 시험 볼 때 "왜 이 답 골랐는지" 이유 5가지 적는 것과 같아요.

**색깔과 막대 읽는 법**:
- **초록색 글자 (BULLISH)** = "이 신호는 오른다는 뜻" (좋은 소식)
- **분홍색 글자 (BEARISH)** = "이 신호는 떨어진다는 뜻" (나쁜 소식)
- **막대 길이** = 얼마나 중요한 이유인지 (길수록 중요)

---

#### 각 이유 쉽게 풀어보기

| 이름 | 쉬운 설명 | 지금 값 | 방향 |
|------|---------|--------|------|
| `yield_spread_proxy` | 단기/장기 채권 이자 차이 | 0.365 | 오름 신호 |
| `spy_price_vs_50ma_pct` | 50일 평균보다 얼마나 싼가 | -3.049 | 이미 많이 떨어짐 = 내림 신호 |
| `spy_vol_trend_5d` | 최근 거래량 추세 | 0.03 | 살짝 오름 신호 |
| `spy_return_1m` | 최근 한 달 수익률 | -4.277 | 많이 떨어짐 = 내림 신호 |
| `xlu_relative_1m` | 안전 업종(전기) 강세 | 2.309 | 사람들이 겁먹음 = 내림 신호 |

**AI의 투표 결과**:
- 오름 신호 (BULLISH) 2개
- 내림 신호 (BEARISH) 3개
- → **"떨어질 가능성이 더 커요!"** 판정

### 5) MODEL INFO — "예측 로봇의 성적표"

```
┌─────────────────────────────────────────────┐
│ SPY CV ACC   QQQ CV ACC   TRAINED   FEATURES│
│ 53.2%        50.9%        2026-04-05   27   │
│                           0d ago            │
└─────────────────────────────────────────────┘
```

**하나씩 풀어보기**:

| 항목 | 뜻 | 쉬운 비유 |
|------|---|---------|
| **SPY CV ACC 53.2%** | 로봇이 과거 100번 중 53번 맞춤 | 수학 시험 53점 |
| **QQQ CV ACC 50.9%** | 로봇이 과거 100번 중 51번 맞춤 | 수학 시험 51점 |
| **TRAINED 2026-04-05** | 로봇 공부시킨 날짜 | "오늘 새로 배운 로봇" |
| **0d ago** | 며칠 전에 공부했는지 | "0일 전 = 오늘" |
| **FEATURES 27** | 로봇이 보는 힌트 개수 | "27가지 보고 판단해요" |

---

#### 잠깐! 53점이 좋은 거야?

엄마, 아빠한테 "53점 받았어요" 하면 혼날 것 같죠?
하지만 **주식 예측에서는 다른 이야기**예요.

| 비교 | 정답률 |
|------|-------|
| 동전 던지기 (운으로 맞히기) | 50% |
| 이 AI 로봇 | **53.2%** |
| 좋은 투자 로봇 (프로) | 55~60% |
| 100% 맞히는 로봇 | 없음 (존재 불가) |

**50%를 조금이라도 넘으면 대단한 거예요!** 
주식 예측은 **날씨 예측보다 훨씬 어렵거든요**. 
(비는 구름 보면 알지만, 주가는 사람 마음이 움직이니까요.)

---

### 6) PREDICTION HISTORY — "지금까지 예측 기록장"

```
DATE        SPY    SPY PROB   QQQ    QQQ PROB   MODEL ACC
2026-04-05  ↓ DN   8.5%       ↓ DN   13.8%      53.2%
2026-04-05  —      —          —      —          0.0%
```

**이게 뭐예요?**
- 매일 로봇이 예측할 때마다 여기 **기록돼요**.
- 마치 **숙제 공책**에 매일 답 써놓는 것과 같아요.
- 나중에 **"저번 주 예측이 맞았나?"** 확인할 수 있어요.

**위쪽 그래프**:
- 초록색 선 = SPY가 오를 확률 변화
- 노란색 선 = QQQ가 오를 확률 변화
- 점선 (가운데 0.5) = "반반 (모르겠다)"

그래프가 **점선 위**면 "오를 것 같다", **점선 아래**면 "떨어질 것 같다".
지금 두 선 모두 아래쪽으로 내려가고 있어요 → **하락 예측 강해짐**.

## 다른 페이지들도 살짝 소개

### Overview — "시장 종합 건강검진서"

**병원 건강검진 결과표** 같은 거예요.

화면에 이렇게 나와요:
- **MARKET REGIME: NEUTRAL** = 시장 상태 "보통" (노란불 신호등)
- **CONFIDENCE: 60%** = "이 진단 60% 확신해요"
- **SCREENED: 20** = "20개 주식 조사함"
- **AI ANALYZED: 10** = "그중 10개는 AI가 자세히 분석"
- **CAUTION** (노란 박스) = "조심해서 투자하세요!" 경고
- **TOP 5 ALPHA PICKS** = 오늘 살만한 주식 Top 5

---

### Market Regime — "시장 체온계 5개"

병원에서 **체온계, 혈압계, 심전도** 여러 검사 하듯, 시장도 5개 센서로 측정해요.

| 센서 | 비중 | 뭘 재는가? | 비유 |
|------|-----|---------|------|
| **VIX** | 30% | 공포 수준 | 체온계 (열나면 위험) |
| **TREND** | 25% | 오름세 / 내림세 | 키 재기 |
| **BREADTH** | 18% | 몇 종목이 오르는가 | 반 평균 점수 |
| **CREDIT** | 15% | 위험한 대출 인기? | 모험심 |
| **YIELD CURVE** | 12% | 장기/단기 이자 차이 | 미래 예측 |

5개가 투표해서 **"위험해!" (RISK OFF)** / **"보통"** (NEUTRAL) / **"안전해!"** (RISK ON) 판정.

지금 화면은:
- 전체 판정: **NEUTRAL** (보통)
- VIX만 RISK OFF (공포 살짝 높음)
- 나머지 4개는 NEUTRAL/RISK ON

→ "전체적으로 보통이지만, 살짝 조심할 때" 라는 뜻.

### Top Picks — "선생님 추천 주식 Top 10"

선생님이 "이 학생 공부 잘해요!" 추천하듯, 시스템이 **살만한 주식 10개**를 추천해줘요.

표 읽는 법:
- **GRADE**: A(최고) → F(최악) 등급
- **INTELLIGENCE SCORE**: 종합 점수 (100점 만점)
- **QUANT / AI / TECH / FUND**: 4가지 점수
- **RS vs SPY**: 시장 평균보다 얼마나 잘했나
- **RECOMMENDATION**: 추천 한마디

**추천 종류**:
| 한국어 | 뜻 |
|-------|-----|
| 적극 매수 | "지금 빨리 사세요!" |
| 매수 | "사도 좋아요" |
| 조정 시 매수 | "살짝 떨어지면 사세요" |
| 보류 | "지금은 기다려" |

화면 예시 (1등):
- **EIX**: 67.6점, B 등급, "조정시 매수"
- → Edison International 회사, B 등급이니까 괜찮음. 살짝 떨어지면 사라.

---

### AI Analysis — "AI 선배의 종목 분석 노트"

AI가 **각 주식마다 자세한 분석 노트**를 써줘요. 5부분으로 나뉘어요.

```
1. INVESTMENT THESIS    "왜 이 주식을 살만한가요?" 요약
2. AI SIGNAL STRENGTH   AI 확신도 (%)  +  추천 (BUY/SELL)
3. BULL CATALYSTS       "이래서 오를 거예요!" 좋은 이유 (초록)
4. BEAR RISKS           "이거 위험해요!" 나쁜 이유 (빨강)
5. DATA CONFLICTS       "이 부분 애매해요" 모순점 (주황)
```

**EIX 주식 AI 분석 예시**:
- Investment Thesis: "6.6% 수익률 기록, B 등급이지만 AI 신호 약함"
- BUY 추천
- Bull (좋음): 2026-27년 긍정적 가이던스, AI 그리드 투자
- Bear (나쁨): 최근 '매도' 등급 하향, 핵심 지표 부재
- Conflicts (모순): 긍정 뉴스와 부정 등급이 섞여 있음

마치 선배가 **"이 학원 추천? 장점은 ○○, 단점은 △△" 정리해주는 노트**와 같아요.

대시보드 상단에 크게 표시되는 **Verdict (판정)**이에요.

| **GO** | "투자하기 좋은 날!" | 초록불 신호등 | 초록 |
| **CAUTION** | "조심해서 투자해" | 노란불 신호등 | 노랑 |
| **STOP** | "지금은 쉬어가자" | 빨간불 신호등 | 빨강 |

**어떻게 결정해?** 3개가 투표해요:
1. **Regime** (시장 체제) — 위험한지 안전한지
2. **Gate** (섹터 신호등) — 11개 업종 점수
3. **ML Predictor** (AI 예측) — 다음 주 오를까?

3개 다 "좋다!" → **GO** / 하나라도 "위험!" → **STOP** / 나머지 → **CAUTION**

### Strategy & Setup — "이 주식은 어떤 패턴?" 전략 분류

각 주식마다 **Strategy (전략)**과 **Setup (진입 패턴)**이 표시돼요.

| 이름 | 뜻 | 비유 |

**Setup (진입 패턴):**

### Brier Score — "확률 정답률 채점표" (곧 추가 예정)

AI가 "70% 확률로 오른다"고 했을 때, 정말 70%만큼 정확한 걸까요?

**점수 채점표**:
| Brier Score | 의미 | 비유 |
|-------------|------|------|
| 0.00 | 완벽 | 모든 확률을 정확히 맞힘 |
| 0.25 | 랜덤 | 동전 던지기 수준 |
| 0.332 | 우리 AI 현재 점수 | 랜덤보다 살짝 나음 |
| 0.239 | **보정 후 목표** | **28% 좋아짐!** |
| 1.00 | 완전 틀림 | 답을 반대로 말함 |

**어떻게 좋아지나요?** **Platt Scaling**이라는 확률 보정 기법을 쓰면 됩니다.

**비유**: 엄마가 "비 올 확률 80%"라고 했는데 실제 5번 중 3번만 비가 왔다면, 엄마의 80%는 사실 60%예요. 이렇게 **말한 확률과 실제 확률을 맞춰주는 게 Platt Scaling**.

**검증된 효과** (Summary2-11 실측):
- 보정 전 Brier 0.332 → 보정 후 **0.239**
- Accuracy는 똑같은데 **확률값만 정확해짐**
- "91.5% 확신" 이라고 하면 **정말 91.5%만큼 맞는** AI가 됨

**왜 중요해?** 신뢰도가 정확해야 **얼마나 과감히 투자할지** 판단할 수 있어요!

**참고 이미지**: Summary2-8 (Brier Score 개념), Summary2-11 (Platt Scaling 비교표)
