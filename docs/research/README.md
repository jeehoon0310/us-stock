# Research

전략 리서치 및 소스 개선 아이디어 관리 폴더.

## 구조

| 폴더 | 용도 |
|------|------|
| `strategies/` | 리서치한 매매 전략 노트 (검토 중 / 적용 완료 / 보류) |
| `improvements/` | 소스 개선 아이디어 및 백로그 (docs/evolution/backlog.md는 실행 완료 사이클 기록, 여기는 실행 전 아이디어) |

## 파일 네이밍

- 전략: `YYYY-MM-DD_전략명.md`
- 개선: `YYYY-MM-DD_개선주제.md` 또는 `backlog.md`에 목록 관리
