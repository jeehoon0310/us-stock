### 1. Freqtrade (48.5K)

**URL**: https://github.com/freqtrade/freqtrade
**별 수**: 48,500

## us-stock 현재 아키텍처 요약

**6-Stage Pipeline**:

1.	Data Collection: yfinance, FRED, Finnhub, Yahoo Finance
2.	Market Regime Detection: 5-sensor voting (VIX 30%, Trend 25%, Breadth 18%, Credit 15%, Yield 12%)
3.	Smart Money Screening: 6-factor composite scoring (Technical 25%, Fundamental 20%, Analyst 15%, RS 15%, Volume 15%, 13F 10%)
4.	AI Analysis: Multi-LLM (Gemini → GPT → Perplexity fallback)
5.	Index Prediction: GradientBoosting, 27 features, TimeSeriesSplit
6.	Dashboard: HTML SPA

**강점**: 자동화 파이프라인, 멀티시그널 마켓타이밍, Regime 적응형 리스크, 비용관리 LLM ($13.50/월)
**한계**: EOD 데이터, 실행 없음, 백테스트 부재, S&P 500만, 옵션/선물 없음

## 즉시 실행 가능한 개선 3가지

### 1. Backtrader 통합

```bash
# 설치
~/.local/bin/uv pip install backtrader --python .venv/bin/python3

# 기존 6-factor 스크리닝 → Backtrader 전략으로 변환
# us-stock의 smart_money_picks → Backtrader에서 3년 백테스트
# 출력: Sharpe Ratio, Sortino Ratio, Max Drawdown, Win Rate
```

Claude Code 활용: "us-stock 스크리닝 결과를 Backtrader 전략으로 변환해줘" → 코드 자동 생성

### 2. 한투 API 연동

```bash
npx @koreainvestment/kis-quant-plugin init --agent claude
```

-	us-stock Verdict가 GO이고 Grade A인 종목 → 한투 API로 모의투자 자동 매수
-	Trailing Stop 시그널 → 한투 API로 자동 매도

### 3. Telegram 알림

-	기존 Telegram MCP 활용
-	매일 실행 후 Verdict + Top 5 Picks를 Telegram으로 자동 전송
-	코드 추가: pipeline 완료 시 Telegram send_message 호출

### 4. LLM Wiki 통합

-	us-stock 분석 결과(Verdict, Top Picks, Regime 변화)를 LLM Wiki에 자동 축적
-	다음 세션에서 "어제 시장 어땠어?"라고 물으면 wiki에서 즉시 답변
-	구현: pipeline 완료 시 wiki_ingest 호출 → 22KB wiki에 당일 분석 결과 추가
