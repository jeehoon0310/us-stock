  - "그래서 12가지 역할을 하면서도 7시간 잡니다"

| | Before (AI 이전) | After (AI 이후) |
|---|---|---|
| 미팅 후 | 밤새 회의록 정리 + 자료 작성 | PLAUD 5분 자동 생성, 프로젝트 자동 분류 |

| 유튜브 리서치 | 영상 보며 수동 메모 | URL 던지면 보고서 자동 생성 |
| 이직 준비 | 퇴근 후 별도 시간 확보 | 면접 준비 에이전트가 자동 분석 |
| 강의 준비 | 일주일 자료 정리 | 8-agent 팀이 3시간에 완성 |

- "중요한 건 미팅이 많다는 게 아닙니다. 예전에는 이 미팅 하나하나가 야근의 원인이었습니다. 지금은 아닙니다."

### 4-1. 투자 태도 "우리는 보통 사람" (7분)

> "천체의 움직임은 계산할 수 있어도, 인간의 광기는 도저히 계산하지 못하겠다." -- Newton, 남해회사(South Sea Bubble) 손실 후

**토킹 포인트:**
- 인구 분포: 80% 보통 / 16% 평균 이상 / 4% 다른 차원(오너)
- 버핏: "Rule 1: 돈을 잃지 마라. Rule 2: Rule 1을 잊지 마라"
- 우리 버전: "Rule 1: 우리는 바보다 (80%). Rule 2: Rule 1을 잊지 마라"
- System 1(원숭이 뇌) vs System 2(논리 뇌) -- Kahneman
  - System 1: 빠르고, 감정적이고, 직관적
  - System 2: 느리고, 분석적이고, 논리적
  - 주식 시장에서 System 1이 돈을 날린다
- **행동장치:** 감정적 결정을 막는 규칙 시스템
  - "감정이 아니라 규칙이 매매한다"

> "생각하고, 기다리고, 금식하라." -- Siddhartha (Hesse), 투자 원칙으로 재해석: 원칙 생각 -> 기회 대기 -> 절제(현금 확보)

- 로드리게 버블 4단계: 잠복 -> 인식 -> 광기 -> 폭락
- "택시기사가 주식 얘기하면 팔 때입니다"
- Dunning-Kruger: 조금 배우면 다 안다고 착각
  - "이 강의 듣고 '나 이제 다 안다' 하면 그게 바로 Dunning-Kruger입니다"

### 4-3. 밸류에이션 기초 (7분)

> "주가 = EPS x PER" -- John Burr Williams, Theory of Investment Value (1938)

> "측정되지 않는 것은 관리되지 않는다." -- Drucker (통상 귀속)

**토킹 포인트:**
- 가격 = EPS x PER
  - EPS = 1주당 순이익 (회사가 얼마나 버는지)
  - PER = 시장이 얼마나 기대하는지 (비싸다/싸다)
- 윌리엄스가 1938년에 정의한 공식이 지금도 유효하다
- 드러커의 말처럼 측정해야 관리할 수 있다 -- 재무분석이 필요한 이유
- DuPont ROE = 이익률 x 자산회전 x 레버리지
  - "레버리지로 높은 ROE = 위험. 마진으로 높은 ROE = 품질"
  - 예: 부채로 ROE 30% vs 마진으로 ROE 30% -- 같은 숫자, 완전히 다른 의미
- Fed Model: 이익수익률(1/PER) vs 10년 국채 -- 높은 쪽이 승리
  - S&P 500 이익수익률 5.05% vs 국채 4.3% -> 주식이 약간 매력적
- S&P 500 Forward PER 19.8x (현재)

> [!tip] [v5] 참여 장치 #2 -- 미니퀴즈
> "삼성전자 PER이 12이고 SK하이닉스가 8이면 어디가 더 싼가요? ... 정답: PER만으로는 판단 불가! 업종, 성장률, 부채비율까지 봐야 합니다."

- EPS 매트릭스:

| | PER 상승 | PER 하락 |
|---|----------|----------|
| EPS 상승 | 좋음 | 최고 |
| EPS 하락 | 최악 | 나쁨 |

**시각자료:**
- 가격 = EPS x PER 공식 슬라이드
- DuPont 분해 다이어그램
- Fed Model 비교 차트
- EPS 매트릭스 2x2

### 4-4. 채권 기초 (3분)

> [!info] 블록2->블록3 경계

> "Malkiel's 5 Theorems -- 채권 가격과 금리의 관계 5대 정리" -- Burton Malkiel, A Random Walk Down Wall Street

> "워털루 전투에서 로스차일드가 가장 빨리 정보를 얻어 채권을 매수했다. 이것이 근대 채권 투자의 기원이다." -- 역사적 일화

**토킹 포인트:**
- 채권 가격과 금리는 역의 관계 (시소)
- 말키엘 5법칙 요약:
	1. 가격과 수익률 반대 방향
	2. 만기 길수록 변동성 큼
	3. 만기 길수록 금리 하락 시 수익 큼
	4. 볼록성 존재 (상승 > 하락)
	5. 낮은 쿠폰 + 긴 만기 = 최대 볼록성
- Duration 개념: 금리 1% 변동 -> 채권 가격 이동 %
	- Duration 5 = 금리 1% 상승 시 채권 가격 약 5% 하락
- "채권을 사라는 게 아닙니다. 채권을 이해해야 주식을 이해합니다"
	- Yield Curve가 주식에 미치는 영향
	- 채권 시장은 주식 시장보다 똑똑하다

**시각자료:**
- 채권 가격/금리 시소 일러스트
- 말키엘 5법칙 요약 카드
- Duration 효과 그래프

### 4-5. 위험관리 -- MDD 수학 (8분)

> "손절은 신속히 하라. 이익은 최대화하라. 작게 매매하라(Cut losses short. Ride winners. Keep bets small)." -- Ed Seykota, Market Wizards

**토킹 포인트:**

|------|----------|

  1. Stop Loss: 매수가 -5% -> 자동 매도
  2. Profit Taking: +10% -> 부분 매도
  3. Trailing Stop: 최고가 - N% -> 수익 보호

- 소로스 6단계: 총자본 -> 위험% -> 종목선택 -> 변동성측정 -> 포지션크기 -> 손절
- 승률 vs 손익비:
  - 40% 승률 + 5:1 비율 > 70% 승률 + 1:1 비율
  - "맞추는 횟수보다 맞췄을 때 얼마나 버느냐가 중요합니다"
