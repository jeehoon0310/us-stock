## 3. QA Simulator 리뷰 — 질문 예측

### 빈출 질문 Top 5 요약

| # | 질문 | 핵심 답변 방향 |
|---|------|-------------|
| 1 | "500종목 시스템 어떻게 만들었나요?" | Claude Code + yfinance 조합, 월 $13.50 비용 구조 설명 |
| 2 | "초보자도 따라할 수 있나요?" | 3단계 진입 경로 제시 (유튜브→강의→커뮤니티) |
| 3 | "AI가 주식을 대신 골라주나요?" | "AI는 도구. 판단은 Rule 기반 본인이" 명확히 |
| 4 | "실제 수익률은 어떻게 되나요?" | 면책 선제 후 "백테스트 기준 참고치" 방식으로 답변 |
| 5 | "한투 API 비개발자도 가능한가요?" | S6 축소 반영. "기본 Python 이해 후 가능" |

### 위험 질문 Top 3 요약

| # | 위험 질문 | 대응 전략 |
|---|---------|---------|
| 1 | "이 종목 지금 사도 되나요?" | "추천 불가" 명확히 + 면책 문구 즉시 제시 |
| 2 | "강사님은 지금 뭐 보유하고 있나요?" | 보유 공개 원칙 없음. "포트폴리오 공개는 법적 이슈" |
| 3 | "AI 시스템 수익률 몇 %예요?" | "과거 백테스트 기준이며 실제 수익 보장 없음" + 면책 |

### Q&A 10분 시나리오 요약

- **0-2분**: 종합 질문 수집 (손 들기)
- **2-7분**: 빈출 5개 질문 그룹 처리 (면책 문구 선제 후 답변)
- **7-9분**: 위험 질문 처리 (단호하게 면책 → 교육 관점 전환)
- **9-10분**: "방향 정하고 주무세요" 마무리 + 채널 구독/카카오 오픈채팅 안내
