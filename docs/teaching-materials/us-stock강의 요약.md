## 2. VaR — "최악의 경우 얼마나 잃을까?"

> Value at Risk — 월가 은행도 매일 계산하는 리스크 지표를 우리도 만들어요

### VaR이 뭐예요?

**"95% 확률로, 다음 1주 동안 최대 이만큼 잃을 수 있어요"**

- VaR = **-3.2%** ($3,210)
- CVaR = **-4.6%** ($4,550)

**비유 — 우산**
일기예보에서 "비 올 확률 5%"라고 하면 대부분 우산 안 가져가죠? VaR은 그 5%에서 비가 얼마나 쏟아지는지 알려줘요. CVaR은 "정말 비 오면 폭우급이야"라는 추가 경고예요!

### 어떻게 계산하나요?

```python
# 6개월 수익률에서 하위 5% 찾기

portfolio_returns = 매일 수익률 계산
horizon_returns   = 5일(1주) 합산 수익률

VaR  = 하위 5번째 백분위 값    # "20번 중 19번은 이 안에 있다"
CVaR = VaR보다 더 나쁜 경우들의 평균  # "나머지 1번은 평균 이만큼 잃는다"
```

### Student-t VaR (보너스)

주식 수익률은 정규분포보다 극단적 사건이 더 자주 일어나요. Student-t 분포는 이런 "포리 위험"을 더 잘 잡아내요. scipy가 있으면 자동 계산!

## 3. Drawdown + Stop-Loss — 고점에서 얼마나 빠졌나

> 최고점 대비 하락률을 추적하고, 선을 넘으면 자동 경고!

### Drawdown이 뭐예요?

**NVDA 예시**

| 항목 | 값 |
|------|-----|
| 최고가 (3개월) | $950 |
| 현재가 | $850 |
| Drawdown | **-10.5%** (고점에서 10.5% 빠짐) |
| 고점 이후 일수 | 22일 (22일째 회복 못 함) |

**비유**
시험에서 최고 점수가 95점이었는데 이번에 80점 받았어요. "15점 빠졌네!" = Drawdown -15.8%. 빠진 폭이 크면 뭔가 문제가 있는 거예요.

### Drawdown 심각도

| Drawdown | 심각도 | 비유 |
|----------|--------|------|
| 0 ~ -5% | 정상 | 감기 수준 |
| -5% ~ -10% | 주의 | 독감 수준 |
| -10% ~ -20% | 경고 | 입원 필요 |
| -20% 이상 | 위험 | 중환자실! |

### Stop-Loss — 손절 자동 감시

Part 3에서 추천한 가격(entry_price) 대비 **-8% 이하**로 떨어지면 자동 경고!

**예시**
- DAVE: 추천가 $100 → 현재가 $91 / 변화: **-9.6%** ≤ -8.0% → 🔴 **STOP-LOSS BREACHED!**
- REPX: 추천가 $58 → 현재가 $47 / 변화: **-6.0%** > -8.0% → 아직 안전

**Part 5 연동**
Part 5의 ML이 "다음 주 하락"이라고 예측하면, stop-loss 기준을 -8%에서 -6.5%로 타이트하게 자동 조정해요. 더 빨리 경고!

## 4. 집중도 + 상관관계 — 달걀을 한 바구니에?

> "분산 투자" 제대로 되고 있는지 자동으로 체크해요

### 섹터 집중도

15개 종목 중 Technology가 7개 = **46.7% → 경고!**

| 섹터 | 비율 | 상태 |
|------|------|------|
| Technology | 7/15 = **46.7%** | ⚠️ 경고 (40% 초과) |
| Healthcare | 3/15 = 20.0% | ✅ OK |
| Energy | 2/15 = 13.3% | ✅ OK |

**비유**
시험에서 수학만 공부하면 수학 시험 망하는 날 전체 성적이 폭락해요. 여러 과목을 고루 공부해야 한 과목이 망해도 버틸 수 있어요. 주식도 똑같아요!

### 상관관계 분석

두 종목이 항상 같이 움직이면 분산 효과가 없어요!

| 종목 쌍 | 상관계수 | 판정 |
|---------|---------|------|
| NVDA ↔ AMD | **0.89** | 거의 쌍둥이! 위험 |
| AAPL ↔ MSFT | **0.82** | 높은 상관. 위험 |
| NVDA ↔ XLE | 0.15 | OK (반도체 vs 에너지) |

> 상관계수 **0.80 이상 = 위험**

**비유**
NVDA와 AMD를 둘 다 사는 건 우산 2개 가져가는 것과 같아요. 비 오면 둘 다 펼치지만, 핵폭탄이면 둘 다 쓸모없어요. 차라리 **우산 1개 + 선크림 1개** (서로 다른 종목)가 더 안전해요!
